# Tracking Students Engagement in Canvas

A Python-based analytics tool for tracking and visualizing student engagement within the [Canvas LMS](https://www.instructure.com/canvas) platform.

## Overview

This project analyzes student engagement data exported from Canvas, providing insights into how students interact with course materials. It was developed in the context of an **Ethics in Software Engineering** course and tracks metrics such as:

- **Times Viewed** — how many times each student viewed course content
- **First Viewed / Last Viewed** — timestamps of initial and most recent access
- **Start Date** — when students began engaging with specific modules
- **Per-Student Engagement** — individual breakdown by student name, ID, and cohort

## Visualizations

The notebook generates charts including:

- **Total Times Viewed by Each Student** — a bar chart comparing engagement levels across all students
- Additional engagement distribution and trend plots

## Getting Started

### Prerequisites

- Python 3
- [Google Colab](https://colab.research.google.com/) (recommended) or a local Jupyter Notebook environment

### Running the Notebook

1. Clone the repository:
   ```bash
   git clone https://github.com/True-African/Tracking_Students_Engagement_in_Canvas.git
   ```
2. Open `Untitled1.ipynb` in Google Colab or Jupyter Notebook.
3. Run all cells to generate the engagement analytics and visualizations.
